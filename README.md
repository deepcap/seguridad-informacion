# 🔎 Auditoría TI

Checklist de auditoría de seguridad:

- Revisión de accesos privilegiados
- Validación de MFA
- Segregación de funciones
- Revisión de logs
- Control de cambios
- Gestión de incidentes

### Evidencia común

| Evidencia              | Descripción           |
| ---------------------- | --------------------- |
| Logs de acceso         | Actividad de usuarios |
| Reporte de accesos     | Permisos en sistemas  |
| Políticas de seguridad | Normativa interna     |
| Inventario de activos  | Equipos y software    |

---

# 👤 Gestión de Identidades

Controles clave:

- MFA habilitado
- Contraseñas robustas
- Principio de menor privilegio
- Revisión periódica de accesos

### Roles administrativos

| Rol            | Descripción          |
| -------------- | -------------------- |
| Global Admin   | Control total        |
| Security Admin | Gestión de seguridad |
| User Admin     | Gestión de usuarios  |
